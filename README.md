# 🏥 Hospital Management System (HMS)

A complete Hospital Management System developed using **Python, Django, Bootstrap 5, HTML, CSS, JavaScript, and SQLite**.

The project provides role-based access for **Admin**, **Doctor**, and **Receptionist** with complete hospital management features including patients, doctors, appointments, billing, pharmacy, laboratory, prescriptions, reports, analytics, and PDF export.

# ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/abhishek004punia/Hospital-Management-System-Django.git
```

Move to project

```bash
cd Hospital-Management-System-Django
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py migrate
```

Start Server

```bash
python manage.py runserver
```

Open Browser

```
http://127.0.0.1:8000/
```
